Accessing and Using the Slide-Out Menu

Access the slide-out menu by selecting the “hamburger” icon (three white, horizontal lines) in the upper-right corner of the screen.

Once opened, the slide-out menu features several options:

  • The Menu option will return you to the course menu.
  • The Glossary option will open a glossary of terms included in the course.
  • The Resources option will open a list of resources that are relevant to the course. Some of the resources are links to other Web sites; others are reference documents in Portable Document Format (PDF).
  • The Help option will open a list of frequently asked questions. Select a question to reveal its answer.
  • You should always use the Exit option to exit the course. If you merely close your Web browser, your progress in the course may be lost.

Select the slide-out menu icon again to close this menu.
